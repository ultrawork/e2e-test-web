# E2E-сценарии: Web Notes v24

Верификация страницы `/notes` — рендеринг, добавление/удаление заметок, поиск, счётчик.

**Платформа:** Web (Next.js)
**Инструмент:** Playwright

---

## SC-001: /notes page renders initial state correctly

| Поле | Значение |
|------|----------|
| **Тип** | Page rendering |
| **Приоритет** | High |
| **Режим** | automated (Playwright) |

**Предусловия:**
- Нет

**Шаги:**
1. Перейти на `/notes`

**Ожидаемый результат:**
- Заголовок "Notes" отображается
- Поле ввода "New note" видимо
- Кнопка "Add" видима
- Счётчик показывает "Всего заметок: 0"
- Поле поиска с placeholder "Поиск заметок..." видимо

## SC-006: Each note has a working delete button with correct aria-label

| Поле | Значение |
|------|----------|
| **Тип** | Accessibility + delete |
| **Приоритет** | Medium |
| **Режим** | automated (Playwright) |

**Предусловия:**
- Нет

**Шаги:**
1. Перейти на `/notes`, добавить 2 заметки
2. Проверить наличие кнопок удаления с aria-label
3. Удалить первую заметку

**Ожидаемый результат:**
- Каждая заметка имеет кнопку с aria-label "Delete note: {text}"
- После удаления кнопка первой заметки исчезает
- Кнопка второй заметки остаётся
